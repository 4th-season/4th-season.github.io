# Re:Word 관계언어 분석기 대표 언어 데이터 QA

기준일: 2026-07-23
대상: `groups-1.js` ~ `groups-7.js`

## 점검 기준

1. 대표 표현과 유사표현의 화자·의미 방향이 같은가
2. 자동 생성형 어미 오류와 비문이 없는가
3. 동일 유사표현이 여러 대표 표현에 중복되어 검색 결과를 흐리지 않는가
4. 너무 짧은 keyword가 무관한 표현까지 끌어오지 않는가
5. 표현별 대표 상황이 실제 의미 범위를 충분히 구분하는가
6. 오탈자는 검색 보조용으로만 남기고 화면 후보로 노출하지 않는가

## 1차 확인 결과

### A. 명확한 의미 방향 오류

- `RW051 네가 알아서 해`
  - 유사표현 `내가 알아서 할게`는 화자와 의미 방향이 반대임.
  - 삭제 또는 별도 대표 표현으로 분리 필요.

### B. 자동 생성형 비문·어미 오류

확인 예시:

- `별일 아니야요`
- `아무것도 아니야요`
- `그냥 그냥 넘어가`
- `말해요 뭐 해요`
- `설명 안 해요` 형태 외의 기계적 변형
- `이 정도는 해요줄 수 있잖아`
- `네가 이해요해요`
- `그게 뭐 대단한 거야요`
- `다 너 잘되라고 하는 말이야요`
- `후회한다요`
- `사랑하니까 하는 말이야요`
- `너를 위해요서야`
- `내가 아니면 누가 말해요줘`

처리 원칙:

- 실제 사용 가능성이 낮은 비문은 삭제
- 존댓말형은 자연스러운 문장만 유지
- 오탈자 검색은 화면 후보가 아니라 별도 `typos` 데이터로 분리 권장

### C. 유사표현 중복으로 대표 표현 경계가 흐려지는 항목

- `RW001 괜찮아`에 `별일 아니야`가 포함되지만 `RW006 별일 아니야`가 별도 대표 표현으로 존재
- `RW003 아무것도 아니야`에 `별거 아니야`, `별일 아니야`가 포함되지만 각각 별도 대표 표현이 존재
- `RW031 그만하자`, `RW032 여기까지 하자`, `RW036 나중에 얘기하자` 사이에 `여기서 멈추자`, `지금은 말하지 말자` 등이 교차 중복
- `RW051 네가 알아서 해`, `RW054 마음대로 해` 사이에 `알아서 해`가 중복

처리 원칙:

- 별도 대표 표현으로 존재하는 문구는 다른 대표 표현의 aliases에서 제외
- 의미가 겹치더라도 대표 표현 간 이동은 상황 선택 단계에서 처리
- 검색 후보는 최대 3~5개로 줄이고, 같은 문구가 여러 그룹에서 반복 노출되지 않게 함

### D. 지나치게 넓은 keyword

확인 예시:

- `말해`
- `설명`
- `들어`
- `있지`
- `그래`
- `아니`

문제:

- 짧고 흔한 단어가 포함된 정상 문장까지 무관한 대표 표현군으로 끌어올 수 있음.

처리 원칙:

- 2음절 이하 또는 단독 기능어에 가까운 keyword 제거
- 최소 2개 핵심어 조합 또는 완성 표현 중심으로 검색
- 검색 점수에서 keyword 일치는 alias·canonical보다 낮게 유지

### E. 상황 구조의 반복

여러 대표 표현이 서로 다른 말임에도 동일한 4개 장면과 동일한 기능 순서를 반복 사용함.

예:

- `별일 아니야`, `다 지나간 일이야`, `그냥 넘어가`, `별거 아니야`
- `말해도 몰라`, `됐다 내가 말을 말지`, `설명 안 해`
- `누구는 잘하던데`, `남들은 다 해`, `너만 왜 그래`, `다들 하는데`

문제:

- 결과가 표현보다 상황 템플릿에 의해 거의 동일하게 나올 수 있음.
- 대표 언어를 세분한 이유가 결과 화면에서 드러나지 않음.

보완 방향:

- 각 표현마다 최소 1~2개의 고유 장면 또는 기능 우선순위를 부여
- 같은 어휘군 안에서도 `수용`, `축소`, `체념`, `회피`, `압박`, `비꼼`의 차이를 반영

## 수정 순서

1. 명확한 비문과 의미 반대 alias 제거
2. 별도 대표 표현과 중복되는 alias 정리
3. 짧은 keyword 축소
4. 오탈자 데이터를 `typos`로 분리
5. 동일 장면·기능 반복 표현의 결과 차별화
6. 일반 입력 50개로 검색 결과 회귀 테스트

## 현재 판정

- 데이터 양: 충분
- 검색 범위: 넓음
- 자연스러움: 보완 필요
- 대표 표현 간 경계: 보완 필요
- 상황·결과 차별성: 보완 필요
- 즉시 수정 우선도: 높음
