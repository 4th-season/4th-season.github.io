# 기질·성격 자기이해 점검 문항 작성 지침

- 상태: 공개 베타용 예비문항 작성 기준
- 대상: 만 18세 이상 성인
- 문항 형식: 5점 리커트 척도
- 응답 기준: 전혀 그렇지 않다 / 그렇지 않은 편이다 / 보통이다 / 그런 편이다 / 매우 그렇다

## 1. 문항 작성 원칙

1. 한 문항에는 하나의 행동·감정·판단만 묻는다.
2. 추상적 성격평가보다 최근 생활에서 떠올릴 수 있는 장면을 묻는다.
3. 좋은 사람처럼 보이는 답이 분명한 문장을 피한다.
4. 정상·비정상, 성숙·미성숙, 강함·약함을 암시하지 않는다.
5. 정신질환, 성격장애, 트라우마, 애착유형을 추론하는 문항을 넣지 않는다.
6. 직업·학력·결혼·소득·가족형태에 따라 답이 결정되는 문장을 피한다.
7. 특정 문화·종교·정치적 가치관을 옳은 답처럼 제시하지 않는다.
8. 사용자가 실제 경험하지 못했을 가능성이 큰 특수 상황은 묻지 않는다.
9. 최근 상태와 장기 경향이 섞이지 않도록 ‘대체로’, ‘평소’ 같은 표현을 사용한다.
10. 부정문은 짧고 단순하게 쓰며 이중부정은 금지한다.

## 2. 문항 구성 비율

영역별 예비문항 14개를 다음 비율로 작성한다.

- 일상 행동 장면 5개
- 감정·인지 반응 3개
- 관계·공동생활 장면 2개
- 일·학습 장면 2개
- 반대 방향 확인 문항 2개

반대 방향 확인 문항은 단순한 부정문 반복이 아니라, 같은 영역의 반대 행동을 별도 장면으로 묻는다.

## 3. 금지 예시

- 나는 새로운 일을 좋아하고 사람들과도 쉽게 친해진다.  
  이유: 두 개 이상의 특성을 한 번에 묻는다.

- 나는 책임감이 강한 사람이다.  
  이유: 사회적으로 바람직한 자기평가를 유도한다.

- 나는 불안장애가 있을 정도로 걱정이 많다.  
  이유: 진단을 암시한다.

- 나는 절대로 포기하지 않는다.  
  이유: 극단적 표현이며 실제 행동을 왜곡한다.

- 나는 낯선 상황을 피하지 않는 편은 아니다.  
  이유: 이중부정이다.

## 4. 영역 간 경계

### 변화 반응과 위험 민감성

새로운 것을 선호하는 정도와 위험을 살피는 정도는 반대축이 아니다. 새로움을 좋아하면서도 충분히 준비할 수 있고, 새로움을 선호하지 않으면서도 위험에 둔감할 수 있다.

### 관계 반응성과 협력성

관계 반응성은 인정·친밀감·상대 반응에 얼마나 영향을 받는지 묻는다. 협력성은 차이를 조율하고 함께 문제를 해결하려는 태도를 묻는다.

### 지속성과 자기조절

지속성은 지루함·실패·피로 속에서 계속하는 힘을 묻는다. 자기조절은 목표·기준·책임을 세우고 행동을 조정하는 방식을 묻는다.

### 의미지향과 종교성

의미지향은 종교 유무를 묻지 않는다. 상징, 연결감, 장기적 가치, 개인을 넘어선 관점을 살피되 특정 신념을 우월하게 다루지 않는다.

## 5. 문항 검토 체크리스트

- 한 문항에 한 가지 내용만 있는가
- 특정 답이 도덕적으로 더 좋아 보이지 않는가
- 다른 영역 문항으로도 읽히지 않는가
- 만 18세 이상 일반 성인이 이해할 수 있는가
- 직업·학력·가족형태에 과도하게 의존하지 않는가
- 너무 최근의 기분만 묻지 않는가
- 극단적 표현이 없는가
- 부정문이 복잡하지 않은가
- 결과 해석에 실제로 활용할 수 있는가
- 정식 TCI 문항을 복제하거나 유사하게 바꾸지 않았는가

## 6. 데이터 필드

각 문항은 다음 필드를 사용한다.

```json
{
  "id": "tc_cr_01",
  "domain": "change_response",
  "text": "익숙한 방법이 있어도 새로운 방법을 시험해 보는 편이다.",
  "direction": "forward",
  "context": "daily",
  "status": "draft"
}
```

- `id`: 고유 문항 ID
- `domain`: 7개 영역 ID
- `text`: 문항 문장
- `direction`: `forward` 또는 `reverse`
- `context`: `daily`, `cognitive`, `relationship`, `work`, `counter`
- `status`: 초기에는 `draft`

## 7. 공개 전 문항 QA

- ID 중복 없음
- 영역별 14문항 유지
- 역방향 2문항 이상 4문항 이하
- 같은 어휘와 문장 구조의 과도한 반복 없음
- 영역 간 의미 중복 검토
- 문항 길이와 독해 난이도 점검
- 모바일 한 화면에서 읽기 어려운 장문 제거
- 모든 문항에 1점·3점·5점을 넣은 가상 계산 확인
