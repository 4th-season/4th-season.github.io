# CSAT Reading Page Standard

## 목적
수능 국어 독서 페이지를 단순 요약형 해설이 아니라, 학생이 먼저 판단하고 공식 지문·문항과 비교하는 문해력 활동형 페이지로 통일한다.

- csat/reading/2022/september/01/index.html
- csat/reading/2022/september/10/index.html
- csat/reading/2022/september/14/index.html

## 원문 기준
- 공식 문제지와 평가원 정답·해설을 직접 확인한다.
- 문단 수와 문장 순서를 바꾸지 않는다.
- 필요한 핵심 구절만 최소 범위로 인용한다.
- 평가원 해설을 길게 복제하지 않고 학습용 표현으로 재구성한다.
- 원문에 없는 개념·인과·사례를 사실처럼 추가하지 않는다.
- 미리 만든 화살표 구조에 원문을 끼워 맞추지 않는다.

7. 원문 문장을 차례로 읽기
8. 문장별 상세 해설
9. 앞뒤 원문 연결
10. 복습 흐름도
11. 내가 고른 중심 문장과 비교하기
12. 글 전체에서 문단이 하는 역할

## 문항 기준
- 실제 발문과 ①~⑤ 선택지 구조를 기준으로 한다.
- 임의 문항 제작을 금지한다.
- 문항별 선택과 채점을 독립 작동시킨다.
- 선택하지 않고 채점할 때 안내 문구를 표시한다.
- 채점 전 정답과 해설을 DOM상 시각적으로 숨긴다.
- 채점 후 선택지 주장, 원문 대응, 일치·불일치 지점, 오답 유인을 설명한다.

## 통계 기준
- 저장소의 공식 통계 JSON 또는 검증된 자료만 사용한다.
- 통계가 없으면 임의 수치를 만들지 않는다.
- `현재 보유한 자료에는 포함되지 않았습니다` 또는 `선택률 자료 확인 중`으로 표시한다.

## Note 기준
- 링크만 만들지 않고 실제 파일을 생성한다.
- 본문 반복이 아니라 개념 보충, 비교, 적용, 짧은 연습을 제공한다.
- 페이지당 3~4개를 원칙으로 하되 실제로 필요한 경우만 만든다.

## 완료 판정
다음 검수를 모두 통과해야 진행표에서 완료로 표시한다.
- 공식 문항 범위·정답 일치
- 문단 수·순서 일치
- 채점 전 비노출
- 문항별 독립 작동
- 통계 값 일치
- Note 링크 정상
- canonical·breadcrumb 정상
- HTML·JS 문법 정상
- 모바일 가로 넘침 없음
