# CSAT Reading Audit and Repair Plan

## 진행 원칙
부분 보완군부터 시작하여 대규모 수정군, 전면 리빌드군 순서로 진행한다. 한 페이지는 본문·문항·통계·Note·모바일·진행표 검수를 모두 통과해야 완료 처리한다.

## 판정 기준
- A: 18~20개 기준 충족 — 보존 및 정밀 검수
- B: 14~17개 — 부분 보완
- C: 9~13개 — 대규모 수정
- D: 8개 이하 또는 임의 문항 포함 — 전면 리빌드

## 1단계: 부분 보완군

### 2025학년도 수능
- [ ] 1~3번 독서 전략으로서의 밑줄 긋기
- [ ] 4~9번 개화와 과학·기술 수용
- [ ] 10~13번 기계 학습과 확산 모델
- [ ] 14~17번 인터넷 ID와 명예훼손

보완 항목:
- 문단별 학생 활동을 2~3개로 통일
- 원문 문장별 순차 해설 보강
- 앞뒤 문장 연결 이유 명시
- 중심 문장 비교와 문단 역할 추가
- 실제 선택지별 원문 대응·오답 유인 보강
- 통계 JSON과 Note 링크 검수

### 2026학년도 수능
- [ ] 1~3번 독해 능력의 단순 관점
- [ ] 4~9번 법 해석과 보증 계약
- [ ] 10~13번 열팽창 띠와 액추에이터
- [ ] 14~17번 인격의 통시적 동일성

보완 항목:
- `쉽게 말하면`과 흐름도 앞에 원문 문장별 해설 추가
- 중심 문장 찾기 활동과 학생 판단 비교 추가
- 실제 선택지·정답·평가원 해설 재대조
- 음성 기능과 기본 기능의 충돌 검사

### 2027학년도 6월 모의평가
- [ ] 1~3번 SQ3R 읽기 전략
- [ ] 4~9번 노비 인식과 민권 운동
- [ ] 10~13번 정보 비대칭과 정부 개입
- [ ] 14~17번 라플라스 식과 잉크젯 인쇄

보완 항목:
- 문단별 사전 활동 고정
- 원문 문장별 해설과 중심 문장 비교 추가
- 흐름도를 해설 뒤 복습 자료로 이동
- 문항 선택지와 통계·Note 검수

- [ ] 4~9번 헤겔의 변증법과 예술론
- [ ] 10~13번 기축 통화와 환율
- [ ] 14~17번 차량 주위 영상 장치의 원리

- [ ] 1~3번 독서 활동을 통한 소통의 즐거움
- [ ] 4~9번 유서의 특성과 서학 수용
- [ ] 10~13번 불확정 개념과 재량 준칙

- 선택지별 상세 해설과 오답 유인 분석
