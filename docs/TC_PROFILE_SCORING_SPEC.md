# 기질·성격 자기이해 점검 채점 규칙

- 상태: 공개 후보용 내부 기준
- 계산 위치: 사용자 브라우저
- 토큰·외부 API: 사용하지 않음
- 해석 전제: 표준화 규준이 없는 개인 내부 상대 프로필

## 1. 기본 채점

응답 척도는 1~5점이다.

- 정방향: 응답값을 그대로 사용
- 역방향: `최댓값 + 최솟값 - 응답값`으로 환산
- 각 영역: 10문항
- 영역 원점수 범위: 10~50점
- 화면용 환산점수: 0~100점

환산식:

`(영역 원점수 - 영역 최저점) / (영역 최고점 - 영역 최저점) × 100`

이 점수는 일반 인구에서의 백분위나 표준점수가 아니다.

## 2. 개인 내부 상대 위치

7개 영역 환산점수의 평균과 표준편차를 계산한다. 영역별 위치는 개인 내부 평균과의 차이로만 분류한다.

- clearly_higher: 개인 평균보다 10점 이상 높음
- somewhat_higher: 5점 이상 10점 미만 높음
- near_average: 차이가 5점 미만
- somewhat_lower: 5점 이상 10점 미만 낮음
- clearly_lower: 10점 이상 낮음
- balanced: 전체 영역 편차가 매우 작아 상대 순위를 강조하지 않음

이 분류명은 내부 규칙이며 사용자 화면에는 자연어로 변환한다.

## 3. 프로필 균형도

최고 영역과 최저 영역의 차이로 프로필 형태를 보조적으로 표시한다.

- 0~8점: very_balanced
- 8점 초과~16점: balanced
- 16점 초과~28점: mixed
- 28점 초과: contrasted

균형도는 성격의 안정성이나 건강성을 뜻하지 않는다.

## 4. 묶음 계산

기질 묶음:

- change_response
- uncertainty_sensitivity
- relational_responsiveness
- persistence

성격 묶음:

- self_direction
- cooperation
- meaning_orientation

각 묶음은 평균, 표준편차, 최고·최저 차이를 계산한다. 기질과 성격의 평균을 서로 우열 비교하는 문장은 만들지 않는다.

## 5. 영역 간 관계

7개 영역의 모든 2영역 조합 21개에 대해 절대 점수 차이를 계산한다. 이 값은 이후 해석 엔진에서 다음 용도로 사용한다.

- 점수 차이가 작은 두 경향의 공동 해석
- 상위 영역 간 균형 해석
- 대비가 큰 영역의 긴장 관계 해석
- 중복 조합 결과의 우선순위 결정

## 6. 응답 참고도

응답 참고도는 거짓말이나 성실성을 판정하지 않는다. 이번 응답에서 영역 차이를 얼마나 세밀하게 해석할 수 있는지만 나타낸다.

확인 항목:

- 한 응답값의 반복 비율
- 중간 응답 비율
- 7영역 점수 편차

내부 등급:

- good: 기본 해석 가능
- moderate: 세부 차이를 약하게 표현
- limited: 영역 순위와 조합 단정을 억제하고 균형형 안내를 우선

## 7. 오류 차단

다음 조건에서는 결과를 생성하지 않는다.

- 문항 누락
- 허용되지 않은 응답값
- 중복 문항 ID
- 등록되지 않은 영역
- 올바르지 않은 정·역방향 표시
- profile 모드가 아닌 데이터

## 8. 결과 객체

채점 엔진은 다음 정보를 해석 엔진에 전달한다.

- 7개 영역 원점수·환산점수
- 개인 내부 상대 위치
- 영역 순위
- 1위와 2위의 점수 차이
- 최고·최저 점수 차이
- 전체 균형도
- 기질·성격 묶음 통계
- 21개 영역 쌍의 점수 차이
- 응답 참고도
- 정·역채점이 반영된 문항별 결과

## 9. 금지 사항

- 사회적 평균으로 표현하지 않는다.
- 정상·비정상 범위를 만들지 않는다.
- 백분위나 상위 비율로 표시하지 않는다.
- 점수 차이가 작을 때 순위를 단정하지 않는다.
- 기질과 성격의 총점을 합쳐 사람의 수준을 평가하지 않는다.
