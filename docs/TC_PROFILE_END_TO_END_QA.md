# 기질·성격 자기이해 점검 전체 흐름 QA 기준

- 대상 브랜치: `feature/temperament-character-check-v01`
- 공개 상태: 비공개 후보
- 검사 화면: `/rescan/profile.html`
- 자동 시나리오: `/rescan/tests/profile-scenarios.test.html`

## 1. 자동 대표 시나리오

| ID | 시나리오 | 핵심 확인 |
|---|---|---|
| balanced-mid | 7영역 균형형 | 중간 해석, 조합 과잉 생성 방지, 제한적·보통 참고도 |
| change-persistence-high | 변화 반응·지속성 높음 | 새 시도와 꾸준함 조합 선택 |
| change-high-caution-low | 변화 반응 높음·불확실성 민감성 낮음 | 빠른 실행 조합과 대비형 프로필 |
| caution-high-selfdirection-high | 불확실성 민감성·자기조절 높음 | 준비를 계획으로 연결하는 조합 |
| relationship-cooperation-high | 관계 반응성·협력성 높음 | 관계 신호와 조율 조합 |
| persistence-high-selfdirection-low | 지속성 높음·자기조절 낮음 | 오래 지속하지만 방향 설정이 약한 긴장 조합 |
| meaning-cooperation-high | 의미지향·협력성 높음 | 공동 가치와 협력 조합 |
| contrasted-profile | 강한 대비형 | 최고·최저 차이, 조합 2개 제한, 결과 충돌 여부 |

## 2. 자동 통과 조건

모든 시나리오는 다음 공통 조건을 통과해야 한다.

1. 70개 문항이 모두 채점된다.
2. 7개 영역 점수가 생성된다.
3. 7개 영역 해석이 빠짐없이 생성된다.
4. 2영역 조합은 최대 2개다.
5. 결과 섹션에 제목 또는 본문이 비어 있지 않다.
6. 비진단적 자기이해 도구 고지가 포함된다.
7. 시나리오별 예상 균형도·상위 영역·조합 규칙이 일치한다.

## 3. 수동 화면 QA

### 시작 화면

- 만 18세 이상 권장 대상이 표시되는가
- 정식 TCI·표준화 검사가 아니라는 안내가 보이는가
- 응답 임시 저장 방식이 설명되는가
- 시작 전 확인 절차가 키보드로 가능한가

### 문항 진행

- 10문항씩 7단계로 표시되는가
- 단계마다 영역이 섞여 있는가
- 미응답 상태에서 다음 단계가 차단되는가
- 첫 미응답 문항으로 초점 또는 화면이 이동하는가
- 이전 단계로 돌아가도 응답이 유지되는가
- 새로고침 후 같은 탭에서 현재 단계와 응답이 복원되는가
- 역문항임을 사용자가 알아볼 수 있는 표시가 노출되지 않는가

### 모바일

- 320px 화면에서 가로 스크롤이 생기지 않는가
- 응답 버튼의 글자가 잘리지 않는가
- 터치 영역이 충분한가
- 고정 헤더가 문항 제목이나 오류 안내를 가리지 않는가

### 결과 화면

- 7개 점수 막대와 숫자가 일치하는가
- 점수 막대가 0~100 범위를 넘지 않는가
- 균형형 결과에 과도한 유형명이 붙지 않는가
- 대비형 결과에 정상·비정상 판단이 붙지 않는가
- 주요 조합이 최대 2개만 출력되는가
- 기본 영역 해석과 조합 해석이 모순되지 않는가
- 도움·부담·자기점검 문장이 같은 의미를 반복하지 않는가
- 응답 참고도가 성실성·거짓말 판정처럼 보이지 않는가
- 인쇄 또는 PDF에서 카드가 중간에 과도하게 잘리지 않는가
- 처음부터 다시 하기를 누르면 세션 데이터가 삭제되는가

## 4. 결과 문장 검토 기준

다음 표현이 결과에 포함되면 공개를 중단하고 수정한다.

- 당신은 본래 …한 사람이다
- 정상, 비정상, 위험, 장애, 병리
- 평균보다 우수하다 또는 열등하다
- 상위 몇 퍼센트
- 이 직업에 적합하다 또는 부적합하다
- 타고난 성격이 확정되었다
- 정식 TCI와 동일하거나 유사한 결과라는 주장

모든 문장은 현재 응답에서 나타난 상대적 경향, 도움이 될 가능성, 부담이 될 가능성, 상황에 따른 차이를 중심으로 작성한다.

## 5. 공개 전 판정

다음 조건이 모두 충족되어야 홈 목록에 연결한다.

- 자동 대표 시나리오 전부 통과
- 채점 엔진 단위 QA 전부 통과
- 해석 엔진 단위 QA 전부 통과
- 데스크톱·모바일 수동 QA 완료
- 개인정보 안내와 실제 저장 동작 일치
- 결과 문장 금지 표현 점검 완료
- 버전과 수정일 표시
- 초기 공개판·자기이해용 간이 점검 표기

통계적 규준이나 표본 검증이 없으므로 공개 후에도 `표준화 검사`, `신뢰도 검증 완료`, `정확한 성격 진단`과 같은 표현은 사용하지 않는다.
