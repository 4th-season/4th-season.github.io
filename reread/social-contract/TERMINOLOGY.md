# 사회계약론 Re:Read 번역 용어표 v1.0

이 문서는 장별 번역에서 핵심 정치철학 용어를 일관되게 사용하기 위한 기준이다. 문맥상 자연스러운 한국어가 필요할 때에도 아래 기본 번역어를 우선 검토한다.

| English | 기본 번역 | 사용 원칙 |
|---|---|---|
| social contract / social compact | 사회계약 | 책 제목과 핵심 계약 개념은 `사회계약`으로 통일한다. 장 제목 `The Social Compact`도 사회계약으로 번역한다. |
| compact | 계약 / 결합 약정 | 독립적인 핵심 개념일 때 계약. 일반적인 약정 의미는 문맥에 따라 조정한다. |
| state of nature | 자연 상태 | 역사적 실제 단계라고 단정하지 않는다. |
| state of society | 사회 상태 | 자연 상태와 대비되는 상태를 가리킬 때 사용한다. |
| civil state | 시민 상태 | 단순한 `문명 상태`로 바꾸지 않는다. |
| body politic | 정치체 | 국가, 정부와 구분한다. 계약으로 형성된 집합적 인격을 뜻한다. |
| State | 국가 | 정치체가 수동적·제도적 관계에서 불릴 때 문맥을 확인한다. |
| Sovereign | 주권자 | 군주 개인을 뜻하지 않는다. 일반의지를 행사하는 집합체다. |
| sovereignty | 주권 | 정부 권력과 구분한다. |
| government | 정부 | 주권자의 의지를 집행하는 중간 기관이다. |
| prince | 군주 / 통치자 | 군주정의 지배자는 군주, 일반적 집행 권력 담당자는 문맥에 따라 통치자. |
| people | 인민 | 루소의 정치적 집합 주체를 가리킬 때 `국민`보다 인민을 우선한다. |
| citizen | 시민 | 주권에 참여하는 구성원이라는 능동적 의미. |
| subject | 신민 | 법과 국가 권위에 복종하는 수동적 지위. 같은 개인이 시민이자 신민일 수 있다. |
| association | 결사 / 결합체 | 계약 행위는 결사, 형성된 조직은 결합체로 조정할 수 있다. |
| associate | 결사 구성원 / 결합한 사람 | 문장의 자연스러움에 따라 조정한다. |
| general will | 일반의지 | 붙여 쓰며 고정한다. `보편의지`를 사용하지 않는다. |
| will of all | 전체의 의지 | 일반의지와 명확히 구분한다. 개인 의지의 단순 합이다. |
| particular will | 특수의지 | 개인·집단의 부분적 이해와 연결된 의지. |
| common interest | 공동 이익 | 일반의지의 지향점으로 사용한다. |
| common good | 공동선 | 도덕적·정치적 공통 목적을 강조할 때 사용한다. |
| public force | 공적 힘 / 공공의 힘 | 물리적 강제력만을 뜻하지 않는 경우 `공적 힘`을 우선한다. |
| common force | 공동의 힘 | 사회계약의 핵심 정식에서 사용한다. |
| law | 법 | 일반의지가 일반적 대상을 향해 내린 규칙. 명령·법령과 구분한다. |
| legislator | 입법자 | 법을 작성·제안하지만 주권을 대신하지 않는 비상한 역할. |
| legislation | 입법 / 입법 체계 | 행위는 입법, 제도 전체는 입법 체계. |
| magistrate | 집정자 / 행정관 | 고대·로마 맥락은 집정자 또는 행정관. 현대적 `판사`로 오역하지 않는다. |
| administration | 행정 / 통치 | 정부 집행 기능은 행정, 정치 운영 전반은 통치. |
| political right | 정치적 권리 / 정치법 원리 | 부제에서는 `정치법 원리`를 우선 검토한다. |
| right | 권리 / 정당한 권원 | 법적·도덕적 권리는 권리, 지배의 근거는 정당한 권원으로 풀 수 있다. |
| force | 힘 / 무력 | 논증상의 힘은 힘, 군사·강제 맥락은 무력. |
| liberty | 자유 | freedom과 특별히 구분할 필요가 없는 한 자유로 통일한다. |
| natural liberty | 자연적 자유 | 자연 상태의 무제한적 행위 가능성. |
| civil liberty | 시민적 자유 | 법과 공동체 안에서 보장되는 자유. |
| moral liberty | 도덕적 자유 | 충동의 지배에서 벗어나 스스로 세운 법에 복종하는 자유. |
| equality | 평등 | 형식적 동일함이 아니라 정치적 조건의 평등을 포함한다. |
| property | 재산 / 소유권 | 대상은 재산, 권리는 소유권. |
| possession | 점유 | 정당한 소유권과 구분한다. |
| alienate | 양도하다 | 주권·자유·권리를 넘기는 의미. `소외시키다`로 번역하지 않는다. |
| inalienable | 양도할 수 없는 | 주권의 핵심 성격. |
| indivisible | 분할할 수 없는 | 주권의 핵심 성격. |
| convention | 합의 / 협약 | 최초의 정치적 성립 근거는 합의. 구체적 협정은 협약. |
| consent | 동의 | 명시적·묵시적 동의를 모두 포함한다. |
| pact | 협약 / 계약 | 사회계약을 가리키면 계약, 개별 정치 협약이면 협약. |
| slavery | 노예제 / 노예 상태 | 제도는 노예제, 개인 상태는 노예 상태. |
| despotism | 전제정 / 전제 지배 | 정부 형태는 전제정, 행위·관계는 전제 지배. |
| democracy | 민주정 | 현대 민주주의 일반과 구분하기 위해 정부 형태는 민주정. |
| aristocracy | 귀족정 | 세습·선거·자연적 귀족정을 문맥에 따라 구분한다. |
| monarchy | 군주정 | 군주 개인과 정부 형태를 구분한다. |
| mixed government | 혼합 정부 | `혼합정체`보다 본문 용어와 맞춰 혼합 정부를 기본으로 한다. |
| civil religion | 시민 종교 | 고정 번역어. |
| tribunate | 호민관제 | 기관은 호민관제, 담당자는 호민관. |
| dictatorship | 독재관제 | 로마의 비상 직책을 뜻하므로 현대의 독재 체제와 구분한다. |
| censorship | 감찰관제 | 제4권의 로마적 제도 맥락. 현대의 검열과 구분한다. |
| comitia | 민회 | 로마의 시민 집회. 필요 시 첫 등장에 `로마 민회(comitia)` 병기. |

## 문체 원칙

1. 문장은 현대 한국어로 읽히되 루소 논증의 조건·대조·귀결을 생략하지 않는다.
2. `sovereign`을 문맥 확인 없이 군주로 번역하지 않는다.
3. `government`와 `State`, `Sovereign`, `body politic`을 서로 바꾸어 쓰지 않는다.
4. 한 영문 문장이 지나치게 길면 한국어를 여러 문장으로 나눌 수 있지만 논리 접속은 보존한다.
5. 해설을 번역문에 섞지 않는다. 필요한 설명은 별도 편집자 주로 둔다.
6. 번역의 자연스러움보다 개념 구별이 중요한 구간에서는 용어 일관성을 우선한다.
