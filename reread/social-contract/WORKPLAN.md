# 사회계약론 Re:Read 작업계획

## 작업 대상

장 자크 루소, 『사회계약론』(Du contrat social, 1762)

- 영문 기준본: G. D. H. Cole 번역본(1913), Project Gutenberg·Wikisource 공개본
- 한국어: Re:Read 자체 번역
- 구성: 서문 + 제1권~제4권

## 편집 기준

1. 영문 원문의 장·문단 순서를 고정한다.
2. 각 영문 문단의 첫 문장과 한국어 시작 문장을 직접 대조한다.
3. 영문 문단의 마지막 의미와 한국어 종결점을 함께 확인한다.
4. 한국어는 의미 단위에 따라 여러 문단으로 나눌 수 있으나 하나의 영문 대응 블록 안에 둔다.
5. 각주·번역자 주·출처 표기는 본문 대응 문단으로 계산하지 않는다.
6. 자동 길이 정렬은 초안 생성에만 사용하고, 최종 경계는 의미 대조로 확정한다.
7. 번역에서 누락·중복·과도한 의역이 확인되면 한국어 문장을 조정한다.

## 파일 구조

- `index.html`: 전체 목차
- `preface.html`: 서문
- `book-01-chapter-01.html` 형식으로 장별 저장
- `assets/style.css`: 군주론 Re:Read와 동일한 대응판 UI
- `paragraph-mapping-report.json`: 문단 수·첫 문장·마지막 문장 검증 기록
- `scripts/remap_social_contract.py`: 재생성 및 검증 도구

## 작업 순서

1. 영문 공개본 수집 및 장 구조 확정
2. 장별 영문 HTML 분리
3. 한국어 초벌 번역 작성
4. 첫 문장·마지막 문장 기준 문단 대응
5. 각주와 편집자 삽입문 분리
6. 전 장 검증 보고서 생성
7. 대표 장 수동 검수 후 전체 재검수
8. 목차·내비게이션·메타데이터 연결

## 완료 조건

- 모든 장에 영문·한국어 대응 블록 존재
- 빈 한국어 또는 빈 영문 블록 없음
- 각주가 독립 대응 문단으로 등장하지 않음
- 장별 첫 문장과 마지막 문장 검증 기록 존재
- 이전·목차·다음 링크 전수 정상
- 모바일에서 한국어 본문과 접이식 영문 원문 정상 표시
