- Wikisource 1913 Everyman's Library 스캔
- 장 제목, 권 구분, 각주와 편집자 삽입문의 위치를 확인할 때 사용한다.

### 편집 원칙

1. Cole 번역의 본문 문장을 임의로 현대화하지 않는다.
2. Project Gutenberg의 전자텍스트와 1913년 스캔이 다를 경우 스캔 판면을 우선 확인한다.
3. G. D. H. Cole의 장문 서론은 루소 본문과 분리하며 Re:Read 본문 대응판에 포함하지 않는다.
4. FOREWORD는 `preface.html`로 독립 저장한다.
5. 각주·편집자 주·출처 표기는 해당 본문 문단 안의 `source-note`로 보존하되 독립 대응 문단으로 계산하지 않는다.
6. 한국어는 Re:Read 자체 번역으로 작성한다.
