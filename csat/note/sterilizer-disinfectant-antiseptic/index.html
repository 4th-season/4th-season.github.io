멸균제·감염방지제·소독제

세 용어의 차이는 화학 물질의 이름보다 어디까지 사멸시키는가와 어디에 사용할 수 있는가에서 드러납니다.

1. 포함 관계부터 보기

항미생물 화학제 ├─ 멸균제: 포자를 포함한 모든 병원체 파괴 └─ 감염방지제: 포자를 제외한 병원체 사멸 └─ 소독제: 독성이 비교적 약해 피부·상처에 사용 가능

2. 세 범주의 차이

구분사멸 범위사용 대상핵심 판단
멸균제포자 포함 모든 병원체강한 처리가 필요한 기구·환경범위가 가장 넓음
감염방지제포자를 제외한 병원체물체나 환경포자에는 효과가 제한될 수 있음
소독제감염방지제 범위 안피부나 상처독성이 비교적 약한 하위 범주

3. 지문에서 필요한 이유

이 분류는 ‘어떤 물질이 어떤 구조를 손상시키는가’와 다른 기준입니다. 멸균제·감염방지제·소독제는 효과의 범위와 사용 가능성을 나누고, 알코올·알킬화제·산화제는 작용 방식을 설명합니다.

4. 헷갈리기 쉬운 지점

소독제는 별도 대등 범주가 아니다

소독제는 감염방지제와 나란히 놓이는 범주가 아니라 감염방지제 안에 포함됩니다.

인체 사용 가능과 완전한 안전은 다르다

피부나 상처에 쓸 수 있어도 사람 세포막을 손상시킬 수 있으므로 점막에는 주의해야 합니다.

포자와 일반 병원체를 구분한다

포자까지 제거해야 ‘멸균’이라고 할 수 있습니다.

5. 수능 오답 방식

  • 소독제를 감염방지제와 별개 범주로 제시한다.
  • 감염방지제가 포자까지 모두 제거한다고 바꾼다.
  • 피부에 쓸 수 있다는 이유로 점막에도 안전하다고 확대한다.
  • 작용기제와 사용 범위를 같은 기준처럼 섞는다.
확인 문제

문제: ‘피부에 사용할 수 있으며 포자를 포함한 모든 병원체를 제거한다’는 설명은 왜 부적절한가?

정답: 피부 사용 가능성은 소독제의 특징이고, 포자까지 제거하는 것은 멸균제의 특징이므로 서로 다른 범주의 특징을 합친 설명입니다.
