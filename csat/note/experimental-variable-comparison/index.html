가상 실험 변수 비교

가상 실험 선택지는 한 문장 안에 효과와 안전성처럼 서로 다른 결과를 묶어 제시합니다. 따라서 기준 물질과 변수를 분리해 읽어야 합니다.

1. 비교 순서

  1. 기준이 되는 물질을 먼저 정합니다.
  2. 각 물질에서 달라진 기능만 표시합니다.
  3. 각 기능이 방역 효과와 안전성 중 어디에 영향을 주는지 연결합니다.
  4. 선택지의 결과를 하나씩 따로 검증합니다.

2. 변수와 결과 연결

기능직접 연결되는 결과주의점
지질 손상지질 피막 바이러스 방역 효과, 인체 세포막 안전성효과가 커지면 안전성은 낮아질 수 있음
캡시드 손상바이러스 방역 효과인체 세포막 안전성과 직접 같은 변수는 아님

3. 지문 표를 읽는 방식

A를 기준으로 설정 ↓ B·C·D에서 달라진 기능 표시 ↓ 기능별 결과를 각각 연결 ↓ 선택지의 두 주장 모두 맞는지 확인
예: C가 B보다 지질 손상과 캡시드 손상이 모두 강하다면 방역 효과가 더 클 수 있습니다. 하지만 지질 손상도 더 강하므로 인체 안전성이 같다고 단정할 수 없습니다.

4. 헷갈리기 쉬운 오답

  • 한 기능만 비교하고 전체 효과를 판단한다.
  • 방역 효과가 커졌다는 이유로 안전성도 커졌다고 본다.
  • ‘같다’와 ‘더 강하다’를 표에서 반대로 읽는다.
  • 선택지 앞부분만 맞으면 문장 전체가 맞다고 판단한다.

5. 다시 만났을 때의 기준

선택지에 ‘효과는 커지고 안전성은 같다’처럼 결과가 두 개 나오면, 문장을 둘로 나누십시오. 첫째 결과와 둘째 결과가 모두 근거와 일치해야 전체 선택지가 맞습니다.

확인 문제

문제: X가 Y보다 캡시드 손상은 강하지만 지질 손상은 같다면 무엇을 판단할 수 있는가?

정답: 바이러스 방역 효과는 X가 더 클 가능성이 있지만, 지질 손상과 연결된 인체 안전성은 두 물질이 같다고 판단할 수 있습니다.
