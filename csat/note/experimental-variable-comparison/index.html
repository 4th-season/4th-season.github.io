가상 실험 변수 비교

  1. 기준 물질을 먼저 정합니다.
  2. 각 물질에서 강해지거나 약해진 기능만 표시합니다.
  3. 방역 효과와 안전성에 연결되는 기능을 따로 적습니다.
  4. 두 물질을 비교할 때 바뀐 기능이 하나인지 둘인지 확인합니다.
핵심: 지질 손상 기능은 피막 바이러스 방역 효과와 인체 안전성에 함께 영향을 줍니다. 캡시드 손상 기능은 바이러스 방역 효과에는 영향을 주지만 인체 세포막의 안전성과 직접 연결되지는 않습니다.
