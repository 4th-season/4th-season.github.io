행정규칙

행정규칙은 행정부 내부의 조직과 사무 처리 기준을 정하는 규칙입니다. 원칙적으로 일반 국민에게 직접 적용되지 않지만, 지문에서는 첨단 기술처럼 전문성과 신속성이 매우 필요한 경우 규제 사항을 정하는 예외도 설명합니다.

왜 필요한가

행정부가 내부 업무를 일관되게 처리하려면 조직의 직제, 업무 분담, 처리 절차를 정할 기준이 필요합니다. 이때 행정규칙은 내부 운영의 기준이 됩니다.

원칙과 예외

구분내용
원칙행정부 내부의 직제와 사무 처리 절차를 정함
적용 대상원칙적으로 행정기관 내부
절차대체로 입법예고와 공포가 필요하지 않음
예외첨단 기술처럼 국회가 세부 내용을 예측하기 어렵고 신속한 대응이 필요한 경우 규제 사항을 정할 수 있음

지문에서의 구조

행정규칙은 내부 기준이 원칙 ↓ 일반 국민에게 직접 적용되지 않음 ↓ 그러나 첨단 기술 분야에서는 신속성과 전문성이 필요 ↓ 예외적으로 규제 사항을 정할 수 있음

위임명령과 무엇이 다른가

위임명령은 법률의 구체적 위임을 받아 일반 국민에게 적용되는 세부 규제를 정합니다. 행정규칙은 원칙적으로 내부 기준이며, 예외적인 경우에만 규제 사항을 정할 수 있습니다.

수능에서 만드는 오답

  • 행정규칙은 언제나 일반 국민에게 직접 적용된다고 바꾸기
  • 행정규칙도 반드시 입법예고와 공포를 거쳐야 한다고 일반화하기
  • 예외적 규제 가능성을 모든 분야에 확대하기
  • 위임명령과 행정규칙의 위임 범위가 같다고 처리하기
짧은 확인 문제

문제. 어떤 규칙이 행정기관 내부의 업무 분담만 정하고 있다면, 일반 국민에게 직접 의무를 부과한다고 볼 수 있을까요?

정답. 원칙적으로 그렇지 않습니다. 내부의 직제와 사무 처리 기준을 정하는 행정규칙의 성격에 가깝습니다.
