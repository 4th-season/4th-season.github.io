항미생물 화학제의 작용기제

이 Note는 ‘무엇까지 죽이는가’가 아니라 어디를 어떻게 손상시키는가를 읽는 데 초점을 둡니다.

1. 세 작용 방식 비교

물질주요 표적작용 결과
알코올 화합물지질, 단백질, 세균 세포벽지질 용해, 단백질 변성, 세포벽 약화
알킬화제단백질과 핵산알킬 작용기 결합으로 기능 상실, 복제·발현 교란
산화제캡시드와 내부 단백질산화로 감염력과 효소 기능 상실

2. 표면과 내부를 따로 읽기

표면 구조 손상 ↓ 병원체 보호막 약화 내부로 작용 가능 ↓ 단백질·핵산 기능 저해 ↓ 효소 기능, 복제, 발현이 방해됨

표면 손상과 내부 작용은 완전히 분리된 단계가 아닙니다. 한 화학제가 표면을 손상시키면서 내부 단백질에도 영향을 줄 수 있습니다.

3. 지문에서 중요한 표현

변성

단백질의 구조가 달라져 본래 기능을 잃는 현상입니다.

산화

물질의 화학적 상태를 변화시켜 구조와 기능을 손상시키는 작용입니다.

복제와 발현

핵산의 정보가 복사되고 실제 기능으로 나타나는 과정입니다. 핵산이 손상되면 이 과정이 방해됩니다.

4. 수능 오답 방식

  • 산화제가 캡시드만 손상시킨다고 범위를 축소한다.
  • 알킬화제가 지질만 녹인다고 작용 대상을 바꾼다.
  • 표면 손상과 내부 작용을 서로 배타적인 것으로 제시한다.
  • 단백질 변성과 핵산 복제 저해를 뒤섞는다.
확인 문제

문제: 하이포염소산 소듐이 바이러스 표면의 캡시드와 내부 단백질을 모두 손상시킨다면, ‘표면에만 작용한다’는 판단은 적절한가?

정답: 적절하지 않습니다. 작용 범위를 표면으로만 제한했기 때문입니다.
