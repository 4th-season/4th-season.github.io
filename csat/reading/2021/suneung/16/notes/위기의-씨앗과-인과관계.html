위기의 씨앗과 인과관계

‘위기의 씨앗’은 하나의 사건이 아니라 경제적 번영을 흔들 수 있는 여러 사회·정치적 문제를 묶어 부르는 표현입니다.

1. 공통 배경

급격한 인구 증가는 생계 압박, 이주와 도시화, 관료 경쟁 심화를 낳았습니다. 그러나 뒤에 나타나는 문제들의 직접 원인은 서로 다릅니다.

결사 조직 경로

이주·도시화 → 전통적 유대 약화 → 상호 부조 조직 등장 → 불법 활동·반란 기반 가능성

관료 부패 경로

관료 조직 규모 정체 → 자리 경쟁 심화 → 인맥과 불법 행위 → 부정부패 심화

2. 통치자의 반응

사회 불안과 반란 가능성 + 관료 사회 부패 ↓ 통치자의 불안 확대 ↓ 서양과의 교역을 줄이려는 경향

3. 인과 문제 읽는 법

공통 배경과 직접 원인을 구분해야 합니다. 인구 증가는 두 문제의 넓은 배경이지만, 사회적 유대 약화가 관료 부패를 직접 일으킨다고 쓰면 원문의 경로가 바뀝니다.

4. 수능 오답 방식

번영의 근거인 신용 기관 확대를 위기 징후로 바꾸거나, 민간의 대응을 조정 정책으로 바꾸거나, 서로 다른 두 인과 경로를 하나로 합치는 선택지를 주의합니다.

확인 문제

결사 조직의 불법화와 관료 부패는 같은 직접 원인에서 생겼나요?

정답: 아닙니다. 인구 증가라는 배경은 공유하지만 구체적인 발생 경로는 다릅니다.
