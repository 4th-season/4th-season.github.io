렌더링과 화솟값

렌더링은 모델링으로 만든 3차원 데이터를 특정 관찰 시점에서 보이는 2차원 화면으로 바꾸는 단계입니다.

화소와 화솟값

개념뜻
화소전체 화면을 잘게 나눈 점. 화면에는 정해진 개수의 화소가 사용됨
화솟값각 화소에 부여되는 밝기·색상 등의 값
프레임모든 화소의 화솟값이 결정된 하나의 화면

원근감과 입체감

관찰 시점 설정 → 가까운 물체는 크게, 먼 물체는 작게 표현 → 표면 특성·가려짐·조명·명암·그림자 반영 → 각 화소의 화솟값 결정 → 한 프레임 완성

물체의 색과 질감 같은 표면 특성은 모델링에서 먼저 지정됩니다. 렌더링은 그 값을 재료로 사용해 관찰 시점에 맞는 화솟값을 계산합니다.

가려짐 판단

특정 관찰 시점에서 다른 물체에 가려 보이지 않는 부분은 최종 2차원 화면에 나타나지 않습니다. 따라서 그 가려진 부분의 표면 특성은 해당 화면의 화솟값 계산에 직접 사용되지 않습니다.

수능 오답 방식

‘화솟값이 표면 특성을 결정한다’고 하면 원인과 결과를 거꾸로 읽은 것입니다. 모델링의 표면 특성이 먼저이고, 렌더링의 화솟값이 나중입니다.

확인 문제

화면의 화소 개수는 정해져 있으므로 장면이 바뀔 때 주로 달라지는 것은 각 화소의 화솟값입니다.
