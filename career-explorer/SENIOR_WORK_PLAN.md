# 중장년·시니어 진로탐색 작업계획

마지막 수정: 2026-07-24  
현재 작업 브랜치: `career-senior-s09`

## 1. 운영 목적

중장년·시니어가 지금까지의 경력과 생활조건을 다시 정리하고, 서로 성격이 다른 전환 경로와 직업을 비교하도록 돕는다.

이 프로그램은 직업 적합도, 성공 확률, 취업 가능성을 계산하지 않는다. 경험을 직무능력으로 확정하지 않고 다음 탐색을 위한 연결 근거로 사용한다.

## 2. 운영 경로

- 통합 공개 예정 경로: `/career-explorer/?audience=senior`
- 현재 독립 실행 경로: `/career-explorer/senior/`
- 시작 유형 직접 진입: `/career-explorer/senior/?start={시작유형코드}`

시작 유형 코드:

| 시작 유형 | 코드 |
| --- | --- |
| 경력 이어가기 | `continue` |
| 퇴직 준비하기 | `retire` |
| 다시 취업하기 | `reemploy` |
| 일의 강도 조정하기 | `lighter` |
| 경험 활용하기 | `experience` |
| 새로운 분야 찾기 | `newfield` |

기존 청소년 경로는 변경하지 않는다. S10 검증을 마친 뒤 통합 대문의 `audience=senior` 진입을 독립 실행 경로와 연결한다.

## 5. 유지 원칙

- 나이를 직업 추천·배제 기준으로 사용하지 않는다.
- 경력공백과 미입력을 부족으로 판단하지 않는다.
- 적합도·성공 확률·취업 가능성 점수를 만들지 않는다.
- 경험은 직무능력 확정이 아니라 연결 근거로만 사용한다.
- 필수조건과 충돌 가능성이 있어도 직업을 자동 탈락시키지 않는다.
- 근무시간·임금·지역 수요는 실제 채용공고에서 재확인한다.
- 자격은 법정 필수·담당업무별·기관 및 공고별로 구분한다.
- 개인정보와 선택 내용은 별도 서버에 저장하지 않는다.
- AI 서비스에는 자동 전송하지 않는다.

## 7. S10 검증 계획

### 7-1. 대표 경로 완주

6개 시작 유형을 각각 다음 순서로 실행한다.

1. 시작 유형 선택
2. 경력·경험 입력
3. 생활·근무조건 입력
4. 3개 전환 경로 생성
5. 9개 직업 생성
6. 통합 결과문 확인
7. A4 보고서 미리보기
8. 인쇄·PDF 저장

### 7-2. 입력 경계 사례

- 업종·경력기간·경험을 모두 비운 사례
- 일부 항목만 입력한 사례
- 필수조건이 많은 사례
- 모든 조건을 미결정으로 둔 사례
- 교육 준비기간을 필수로 둔 새로운 분야 사례
- 신체 부담과 근무시간을 필수로 둔 강도 조정 사례

### 7-3. 화면·접근성

- 데스크톱·태블릿·모바일
- 키보드만으로 전체 진행
- 포커스 표시와 버튼 최소 높이
- 뒤로가기·새로고침·초기화
- 긴 업종명과 입력문 처리
- 가로 넘침과 표 잘림
- 인쇄 페이지 분할

### 7-4. 회귀검사

- 기존 청소년 탐색 페이지 변경 없음
- 기존 청소년 직접 진입과 결과 생성 유지
- 시니어 독립 페이지의 스크립트·스타일이 청소년 페이지에 영향을 주지 않음
