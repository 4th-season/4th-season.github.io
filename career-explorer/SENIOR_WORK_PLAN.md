## 1. 운영 목적

중장년·시니어가 지금까지의 경력과 생활조건을 다시 정리하고, 서로 성격이 다른 전환 경로와 직업을 비교하도록 돕는다.

이 프로그램은 직업 적합도, 성공 확률, 취업 가능성을 계산하지 않는다. 경험을 직무능력으로 확정하지 않고 다음 탐색을 위한 연결 근거로 사용한다.

## 2. 운영 경로

- 통합 진입 경로: `/career-explorer/?audience=senior`
- 시니어 직접 경로: `/career-explorer/senior/`
- 시작 유형 직접 진입: `/career-explorer/?audience=senior&start={시작유형코드}`
- 시니어 페이지 직접 진입: `/career-explorer/senior/?start={시작유형코드}`

시작 유형 코드:

| 시작 유형 | 코드 |
| --- | --- |
| 경력 이어가기 | `continue` |
| 퇴직 준비하기 | `retire` |
| 다시 취업하기 | `reemploy` |
| 일의 강도 조정하기 | `lighter` |
| 경험 활용하기 | `experience` |
| 새로운 분야 찾기 | `newfield` |

`audience=senior` 요청만 시니어 페이지로 전달한다. 기존 청소년 기본 경로와 `audience=youth` 이용 흐름은 그대로 유지한다.

## 5. 유지 원칙

- 나이를 직업 추천·배제 기준으로 사용하지 않는다.
- 경력공백과 미입력을 부족으로 판단하지 않는다.
- 적합도·성공 확률·취업 가능성 점수를 만들지 않는다.
- 경험은 직무능력 확정이 아니라 연결 근거로만 사용한다.
- 필수조건과 충돌 가능성이 있어도 직업을 자동 탈락시키지 않는다.
- 근무시간·임금·지역 수요는 실제 채용공고에서 재확인한다.
- 자격은 법정 필수·담당업무별·기관 및 공고별로 구분한다.
- 개인정보와 선택 내용은 별도 서버에 저장하지 않는다.
- AI 서비스에는 자동 전송하지 않는다.
