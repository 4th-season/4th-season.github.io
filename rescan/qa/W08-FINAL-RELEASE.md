# W08 최종 공개 검증

검증일: 2026-07-17

## 공개 범위

- 지각된 스트레스
- 번아웃
- 수면위생
- 자기존중감
- 반추사고
- 관계경계

## 확인 항목

- 홈 검색과 카테고리 필터가 공개 주제 6개를 구분함
- 모든 주제의 데이터 경로와 해설 경로가 index.json에 연결됨
- 문항 ID를 기준으로 계산하여 화면 순서 변화가 점수에 영향을 주지 않음
- 역채점 문항은 척도의 최솟값과 최댓값을 기준으로 계산함
- 결과 구간은 최저점부터 최고점까지 빈틈과 중복 없이 구성함
- 같은 접속 세션에서 문항 순서를 유지함
- 응답값과 점수는 서버나 외부 서비스로 전송하지 않음
- 결과 인쇄와 PDF 저장은 브라우저 기능으로 처리함
- 키보드 초점, 본문 바로가기, 미응답 안내, 자바스크립트 비활성화 안내를 제공함
- 모바일에서 문항 선택지와 검색·분류 도구를 사용할 수 있음
- 각 결과에 진단 대체 불가 고지문을 제공함
- 설계 원칙, 개인정보 안내, 이용 안내를 홈에서 접근할 수 있음
- 사이트맵에 Re:Scan 공개 해설과 안내 페이지를 반영함

## 공개 상태

Re:Scan 1차 공개판 완료. 다음 작업은 신규 주제 확장 또는 기존 문항의 사용자 피드백 기반 재검토로 진행한다.
