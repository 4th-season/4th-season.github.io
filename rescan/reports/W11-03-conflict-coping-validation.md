# W11-03 갈등 대처 검증 보고서

- 대상: `rescan/data/checklists/conflict-coping.json`
- 안내 페이지: `rescan/tistory/conflict-coping.html`
- 판정: 정적 검토 통과

## 구조
- 12문항, 4영역, 영역별 3문항
- `pattern` 모드, 0~36점 자체 설명 구간
- 역채점 문항 7개
- 결과 구간 연속·중복 없음

## 내용 경계
- 회피·격화·사실과 감정의 구분·조정과 회복을 서로 다른 영역으로 구성함
- 특정 갈등 스타일 하나를 모든 상황의 정답으로 제시하지 않음
- 현재 문제와 과거 문제, 사실과 의도 추측, 감정과 요청을 구분함
- 안전을 위협하는 관계에서는 대화 지속보다 거리 두기·기록·중재·외부 지원을 우선함
- `분노조절`과 달리 양측의 차이와 관계 조정 과정에 초점을 둠

## 미검증 범위
전문가 패널 CVI, 사용자 인지면접, 신뢰도, 요인타당도, 규준화는 수행되지 않았다.
