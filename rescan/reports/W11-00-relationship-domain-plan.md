# W11-00 관계와 연결 영역 제작 계획

- 작성일: 2026-07-24
- 대상 프로젝트: Re:Scan `[나 체크리스트]`
- 선행 작업: W10 자기이해 5종 제작·내용 재검토·공개 브라우저 검증 완료
- 현재 공개 체크리스트: 16종

## 1. W11 영역 선정

W11은 **관계와 연결** 영역으로 진행한다.

현재 Re:Scan에는 `관계경계`와 `감정표현`이 공개되어 있으나, 다음 생활 장면은 아직 충분히 다루지 않았다.

- 사람들과 연결되어 있다는 감각과 외로움
- 의견·요구·거절을 공격적이지 않게 표현하는 자기주장
- 갈등이 생겼을 때 회피·격화·조정·회복으로 이어지는 반응
- 한쪽만 주고받는 관계가 아닌 상호성과 책임 균형
- 필요할 때 이용할 수 있는 사회적 지지와 구체적인 도움 요청

W11은 관계의 좋고 나쁨이나 사람의 성격을 판정하지 않고, **최근 생활에서 반복되는 관계 장면과 사용할 수 있는 연결 자원**을 확인하는 방향으로 구성한다.

## 2. 제작 목록

### W11-01 사회적 연결감

- 공개 제목: `사회적 연결감`
- 작업형: 성찰형 `reflection`
- 등급안: B
- 목적: 관계의 수가 아니라 접근 가능한 사람, 소속감, 일상 공유, 고립 뒤 다시 연결하는 행동을 살펴본다.
- 4개 영역:
  1. 연락 가능한 관계
  2. 소속감과 내 자리
  3. 일상 공유와 함께하는 활동
  4. 고립 신호와 재연결
- 주의점:
  - 객관적 사회적 고립과 주관적 외로움을 같은 개념으로 처리하지 않는다.
  - 친구 수·모임 수만으로 연결감을 판정하지 않는다.
  - 심한 외로움에 임상적 절단점을 부여하지 않는다.

### W11-02 자기주장과 의사표현

- 공개 제목: `자기주장`
- 작업형: 패턴형 `pattern`
- 등급안: B
- 목적: 자신의 의견과 필요를 표현하면서도 상대를 공격하거나 통제하지 않는 관계 행동을 살펴본다.
- 4개 영역:
  1. 의견 말하기
  2. 부탁과 도움 요청
  3. 거절과 한계 알리기
  4. 반대 의견 다루기
- 주의점:
  - 자기주장을 공격성·지배성·말의 세기와 동일하게 보지 않는다.
  - 조용한 표현 방식도 명확한 의사표현일 수 있음을 반영한다.
  - `관계경계`와 겹치는 거절 문항은 경계 설정 자체보다 **표현 행동**에 초점을 둔다.

### W11-03 갈등 대처

- 공개 제목: `갈등 대처`
- 작업형: 패턴형 `pattern`
- 등급안: B
- 목적: 가까운 관계에서 갈등이 생겼을 때 회피, 격화, 관점 확인, 조정과 회복이 어떻게 나타나는지 살펴본다.
- 4개 영역:
  1. 회피와 미루기
  2. 격화와 방어
  3. 사실·감정·요구 구분
  4. 조정과 관계 회복
- 주의점:
  - 특정 갈등 스타일 하나를 모든 상황의 정답으로 제시하지 않는다.
  - 안전을 위협하는 관계에서는 대화 지속보다 거리 두기와 외부 지원이 우선일 수 있음을 고지한다.
  - 한 문항에 회피·공격·사과처럼 여러 행동을 함께 넣지 않는다.

### W11-04 관계 상호성

- 공개 제목: `관계 상호성`
- 작업형: 성찰형 `reflection`
- 등급안: B
- 목적: 관심, 시간, 책임, 결정, 돌봄이 한쪽으로만 흐르지 않고 서로 조정되는지 살펴본다.
- 4개 영역:
  1. 주고받는 관심
  2. 책임과 돌봄의 균형
  3. 공동 결정과 조율
  4. 불균형 인식과 조정
- 주의점:
  - 모든 관계에서 같은 양을 주고받아야 한다고 단순화하지 않는다.
  - 질병·돌봄·경제 상황처럼 일시적 불균형이 필요한 조건을 고려한다.
  - 희생의 양보다 의사소통과 합의 가능성을 중심으로 문항을 구성한다.

### W11-05 지지 자원과 도움 요청

- 공개 제목: `지지 자원과 도움 요청`
- 작업형: 점수형 또는 성찰형 중 문항 검토 후 확정
- 등급안: A 또는 B
- 목적: 정서적·정보적·실질적 도움을 받을 수 있다고 느끼는지, 필요한 도움을 구체적으로 요청하고 받아들이는지 살펴본다.
- 4개 영역:
  1. 도움을 구할 사람과 기관
  2. 요청 내용 구체화
  3. 도움 받기와 활용
  4. 요청을 막는 부담감과 불신
- 주의점:
  - 실제 관계망의 크기와 지각된 지지를 구분한다.
  - `자기효능감`과 `회복탄력성`의 자원 활용 문항과 겹치지 않도록 **관계망의 이용 가능성과 지원 경험**에 초점을 둔다.
  - 지원이 부족한 결과를 개인의 의지 부족으로 해석하지 않는다.

## 3. 제외하는 후보

W11에서는 다음 표현을 독립 검사명으로 사용하지 않는다.

- 애착유형
- 관계중독
- 의존성 성격
- 회피형 인간
- 사람 보는 눈
- 독성 관계 판별

이 표현들은 진단·유형 낙인이나 상대방 판정으로 오해될 가능성이 높고, 짧은 자체 문항으로 다루기에는 개념 범위가 지나치게 넓다. 필요하면 향후 `관계 불안`, `거리 조절`, `반복되는 확인 행동`처럼 관찰 가능한 생활 장면으로 다시 나눈다.

## 4. W11 공통 문항 기준

각 체크리스트는 12문항, 4개 영역, 영역별 3문항을 기본으로 한다.

- 최근 2주 또는 최근 4주처럼 응답 기간을 명시한다.
- 한 문항에는 하나의 행동·생각·경험만 넣는다.
- 특정 관계 형태를 전제하지 않고 가족·친구·동료·이웃 등 다양한 관계에 적용되도록 쓴다.
- 성별, 연령, 혼인 여부, 직업, 가족 구성에 따른 고정관념을 넣지 않는다.
- 관계 행동과 감정 결과를 한 문항에서 섞지 않는다.
- 긍정형과 부담형 문항을 함께 쓰되, 역채점이 문장을 어렵게 만들지 않도록 한다.
- 높은 점수를 임상 위험이나 성격 결함으로 해석하지 않는다.
- 결과 구간은 자체 설명 구간이며 규준·절단점이 아님을 명시한다.
- 통제·협박·폭력·스토킹 등 안전 문제가 있는 상황에서는 관계 기술보다 거리 두기와 전문 지원을 우선하도록 안내한다.
