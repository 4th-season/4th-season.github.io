홈주제별 보기이용 안내

PRIVACY

응답 처리 안내

서버 전송 없음

체크리스트 응답과 계산 결과는 별도 서버나 외부 분석 서비스로 전송하지 않습니다. 계산은 현재 브라우저 안에서 이루어집니다.

회원정보 수집 없음

이름, 이메일, 전화번호, 계정 정보와 같은 개인정보를 입력받지 않습니다. 결과 확인을 위해 회원가입도 요구하지 않습니다.

세션 정보

문항 순서를 같은 접속 동안 유지하기 위해 문항 ID 순서만 sessionStorage에 임시 저장할 수 있습니다. 응답 내용이나 점수는 저장하지 않습니다.

인쇄와 PDF

결과 인쇄나 PDF 저장은 사용자의 브라우저 기능으로 처리됩니다. 저장된 파일의 보관과 삭제는 사용자가 직접 관리합니다.

공용 기기에서는 결과 화면을 닫고, 인쇄·PDF 파일을 저장했다면 기기에 남아 있는 파일을 직접 확인해 주세요.
