# Re:Scan 변경 기록

## 2026-07-17 · 1차 공개판

- 공통 체크리스트 엔진과 JSON 데이터 구조 구축
- 문항 ID 기반 계산, 역채점, 영역별 점수와 결과 구간 지원
- 영역 균형형 문항 순서와 세션 내 순서 유지 적용
- 스트레스, 번아웃, 수면위생, 자기존중감, 반추사고, 관계경계 공개
- 주제별 해설과 5문항 간이점검 연결
- 검색·분류 필터, 모바일·키보드 접근성, 인쇄·PDF 저장 지원
- 설계 원칙과 개인정보 안내 공개

## 유지관리 원칙

1. 공개 문항의 문구나 배점을 변경할 때 schemaVersion을 올린다.
2. 문항 ID는 의미가 달라지지 않는 한 재사용하거나 바꾸지 않는다.
3. 문항 수와 척도가 달라지면 결과 구간 전체를 다시 검증한다.
4. 새 주제는 데이터·해설·고지문·왕복 링크·모바일 점검을 모두 마친 뒤 published 상태로 전환한다.
5. 임상 검사나 진단 기준으로 오해될 표현을 사용하지 않는다.
