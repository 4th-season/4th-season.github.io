# W02-D3 계산 엔진 검증 기록

기준일: 2026-07-17

## 검증 결과

- A등급 `scored`: 일반 합산과 영역별 합산 지원
- B등급 `pattern`: 점수 합산 후 패턴형 결과 분기 지원
- C등급 `reflection`: 점수 합산 후 성찰형 결과 분기 지원
- D등급 `help`: 총점과 안정·주의·경계 결과 생성을 차단
- 역채점: `척도 최댓값 + 최솟값 - 원응답` 방식 적용
- 결과 구간: 빈틈·중복·최솟값/최댓값 역전 시 오류 처리

## 샘플 기대값

| 항목 | 입력 | 기대 결과 |
|---|---:|---:|
| 일반 합산 | q01=2, q02=3 | 총점 5 |
| 영역별 합산 | tension=2, recovery=3 | 각 2, 3 |
| 역채점 | 0~3 척도에서 응답 3 | 0점 |
| D등급 | help mode | 점수 결과 차단 |

실제 주제별 문항과 최종 결과 문구는 이후 콘텐츠 작업에서 확정합니다.
