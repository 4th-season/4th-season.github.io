# W04 지각된 스트레스 통합 검증 보고

검증일: 2026-07-17

## W04-D1 근거·영역·문항 기준

- 최근 2주 생활 신호를 기준으로 범위를 제한했다.
- 긴장과 회복 저하 두 영역을 정의했다.
- 공식 자료의 범주만 참고하고 공식 척도 문장은 직접 복제하지 않았다.
- 진단·질환 판정·원인 단정 표현을 배제했다.

## W04-D2 문항

- 정식형: 12문항.
- 영역 배분: 긴장 6, 회복 저하 6.
- 역채점: 긍정형 회복 문항 3개.
- 간이형: 정식형과 동일 문장을 반복하지 않는 5문항.
- 예비문항: 사용하지 않음.

## W04-D3 데이터와 계산

- 총점 범위: 0~36점.
- 결과 구간: 0~8, 9~17, 18~26, 27~36.
- 구간 사이의 빈틈과 중복 없음.
- 문항 순서와 무관하게 고정 문항 ID로 계산.
- 역채점 후 긴장·회복 저하 영역 점수 계산 가능.
- index.json에서 스트레스 상태를 published로 전환.

## W04-D4 결과와 연결

- 비교적 안정, 살펴볼 필요, 부담 누적, 회복 우선의 네 결과 문구 구성.
- 각 결과에 자기질문 2~3개 연결.
- 불편감 지속 또는 일상 기능 영향 시 전문가 상담 고려 문구 포함.
- 내부 해설 글, Re:Scan 이용 안내, WHO 스트레스 자료 연결.

## W04-D5 전체 흐름

- Re:Scan 주제 카드 → 12문항 정식 점검 → 결과 → 해설 글 왕복 이동 가능.
- 해설 글의 5문항 간이점검 → 정식 점검 이동 가능.
- 응답은 브라우저 안에서만 처리하며 외부 저장 기능 없음.
- 모바일 화면, 키보드 선택, 미응답 안내, 다시하기, 인쇄·PDF 저장 기능 유지.
