# Re:Scan W03 점검 기록

## 대상
- 지각된 스트레스 체크리스트
- 공개 경로: `/rescan/checklist.html?id=stress`
- 데이터 버전: `1.2.0`

## 문항 구성
- 총 12문항
- 긴장 6문항
- 회복 저하 6문항
- 긍정형 회복 문항 3개 역채점
- 예비문항 미사용

## 배열 정책
- 영역 균형형 배열
- 같은 영역 최대 2문항 연속
- 같은 브라우저 세션에서 순서 유지
- 새 세션에서 새로운 순서 생성 가능
- 표시 순서와 무관하게 문항 ID로 계산

## 점수 검증
- 응답값 범위: 0~3
- 가능한 총점: 0~36
- 결과 구간: 0~8 / 9~17 / 18~26 / 27~36
- 구간 중복과 누락 없음
- 역채점 후에도 영역별 최고점 동일

## 이용 안전성
- 진단 표현 배제
- 서버 응답 저장 없음
- 결과는 자가점검용 참고 기준임을 명시
- 고득점 결과에서 전문가 상담을 선택지로 안내

## 화면 점검
- 모바일 1열 응답 선택지
- 키보드 포커스 표시
- 진행률과 미응답 안내
- 자바스크립트 비활성화 안내
- 결과 인쇄 및 PDF 저장 지원

## 상태
W03-D2부터 W03-D5까지 통합 점검 완료.
