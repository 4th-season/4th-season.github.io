# Re:Scan W09 검증 기록

- 작업 범위: W09-D1~W09-D5
- 공개 주제: 감정조절, 불안 신호, 우울 신호, 분노조절, 감정표현
- 공통 기간: 최근 2주
- 공통 문항 수: 각 12문항
- 문항 배열: 영역 균형형 무작위 배열, 세션 내 순서 유지

## 데이터 검증

- 감정조절: scored 모드, 4영역, 역채점 4문항, 결과 구간 0~36 연속
- 불안 신호: pattern 모드, 4영역, 역채점 3문항, patternResults 및 호환 results 구성
- 우울 신호: guide 모드, 점수 판정 차단, 안내 단계 제공
- 분노조절: pattern 모드, 4영역, 역채점 4문항, patternResults 및 호환 results 구성
- 감정표현: scored 모드, 4영역, 역채점 3문항, 결과 구간 0~36 연속

## 연결 검증

- rescan/data/index.json에 5개 주제 published 등록
- 각 dataPath와 checklist id 일치
- 각 articlePath 생성
- 정식 점검과 안내 페이지 왕복 링크 적용
- 공식 정보 링크는 WHO와 NIMH 자료를 중심으로 연결

## 안전 문구

- 모든 결과는 진단이 아닌 자가점검 참고 자료로 표시
- 불안·우울 신호는 일상 기능과 지속 여부를 함께 살피도록 안내
- 우울 신호는 점수형 위험 판정을 사용하지 않고 guide 모드로 구성
- 분노조절은 분노 자체보다 표현·회복·관계 영향을 살피도록 구성

## 결과

W09-D1~W09-D5 완료. Re:Scan 공개 주제는 총 11개로 확장됨.
