# W06 자기존중감·반추사고 통합 검증

## 완료 범위
- 자기존중감 4영역 12문항, A등급 점수형
- 반추사고 4영역 12문항, B등급 패턴형
- 두 주제 간이점검 HTML과 정식 점검 연결
- Re:Scan 입구 5개 주제 공개

## 구분 검토
- 자기존중감: 자기평가, 실수 수용, 비교 반응, 자기 돌봄
- 반추사고: 생각 반복, 멈춤 어려움, 생활 영향, 회복 전환
- 자기존중감은 ‘나를 판단하는 기준’, 반추사고는 ‘생각이 반복되는 과정’을 중심으로 구분함

## 검증 결과
- 문항 ID 중복 없음
- 각 주제 12문항, 4영역 각 3문항
- 긍정형 문항 역채점 적용
- 총점 0~36점과 결과 구간 연속성 확인
- 예비문항 사용 안 함
- 결과 문구에서 사람 전체의 가치 판정, 질환 진단, 낙인 표현 배제
- 기존 스트레스·번아웃·수면위생 경로 유지
- 인쇄·다시하기·영역 균형형 문항 순서 공통 엔진 사용

## W07 전달 사항
- 관계경계는 부탁 수용, 책임 과잉, 침범 인식, 관계 후 회복을 중심으로 설계
- 자기존중감의 비교·자기돌봄 문항과 겹치지 않게 관계 장면 중심으로 작성
- 검색·분류 기능 추가 전 기존 5개 카드의 id와 경로를 회귀 검증
